------------------------------------------------------------------------项目简介------------------------------------------------------------------------------------------

简介：
B2C模式的在线教育网站，采用微服务架构，前后端分离开发

主要技术：
前端：Vue，element-ui，nuxt等
后端：Springboot，SpringCloud（gateWay, Feign, Nacos等），Mybatis-plus，JWT, Swagger等
数据库：MySQL, Redis
第三方技术：阿里云OSS，阿里云视频点播，微信支付，用户注册邮箱验证，Git等

功能模块：
前台：热点数据显示，数据分页查询，用户注册与单点登录，订单创建与微信支付，视频点播，
后台：数据库信息增删改查管理，网站数据统计分析

==========================================================================注意===========================================================================================
注意主配置文件MySQL，Redis数据库以及Nacos注册/配置中心，阿里云OSS配置，Spring.mail配置要改成你自己的信息

MyBatis逆向工程中数据库也要换成你自己的信息

MySQL数据库文件为MySQL.sql
